## 👨‍💻 Sobre mim

Sou desenvolvedor **Full Stack** com experiência em **sistemas corporativos**, atuando com **Angular no front-end** e **Java com Spring Boot no back-end**.

Tenho foco em:
- desenvolvimento de interfaces web
- consumo e integração com APIs REST
- implementação de regras de negócio
- manutenção e evolução de aplicações corporativas
- organização de código, performance e manutenibilidade

Atualmente busco evoluir cada vez mais em desenvolvimento web, arquitetura de software e construção de aplicações escaláveis.
